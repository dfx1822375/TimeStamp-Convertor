# Chrome 时间戳转换器 (Timestamp Converter)

一个现代化的 Chrome 浏览器插件，用于快速进行 Unix 时间戳与人类可读日期时间之间的相互转换。

## ✨ 功能特点

*   **双向转换**：支持 时间戳 ↔ 日期时间 的相互转换。
*   **智能识别**：自动识别输入的是秒还是毫秒时间戳，也支持手动指定。
*   **多时区支持**：支持本地时间、UTC 以及全球主要城市（纽约、伦敦、东京、上海等）时区的切换。
*   **毫秒级精度**：默认使用毫秒级精度，满足开发调试需求。
*   **便捷操作**：
    *   一键获取当前时间戳。
    *   一键复制转换结果。
    *   自动格式化日期输入。
*   **现代化 UI**：简洁美观的界面，支持深色模式 (Dark Mode)，流畅的交互动画。

## 🛠 技术实现

本项目是一个标准的 Chrome Extension (Manifest V3)，完全使用原生 Web 技术栈构建，无复杂构建依赖。

### 目录结构
```
e:\Vebe\
├── manifest.json  # 插件配置文件 (V3)
├── popup.html     # 插件弹窗界面结构
├── style.css      # 样式文件 (CSS3, Flexbox, Animation)
├── popup.js       # 业务逻辑 (Vanilla JS)
└── icon.png       # 插件图标
```

### 核心逻辑
1.  **Manifest V3**: 使用最新的 Chrome 扩展标准，更安全、性能更好。
2.  **Vanilla JS**: 不依赖任何第三方库 (如 jQuery, React, Vue)，保持极其轻量 (几 KB)。
3.  **Intl.DateTimeFormat API**: 利用原生国际化 API 处理复杂的时区转换，避免引入庞大的 Moment.js 或 Day.js 库。
4.  **CSS Variables**: 使用 CSS 变量管理主题色和暗黑模式适配。
5.  **交互优化**:
    *   使用 `animation` 实现点击反馈和视觉提示。
    *   监听 `input` 和 `radio` 变化实现流畅的交互体验。

## 📦 安装与使用

1.  克隆或下载本项目到本地。
2.  打开 Chrome 浏览器，访问 `chrome://extensions/`。
3.  开启右上角的 **"开发者模式"**。
4.  点击左上角的 **"加载已解压的扩展程序"**，选择项目根目录。
